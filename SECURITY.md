# Security Policy

## Supported Versions

Security fixes are provided for the latest stable release line.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Use GitHub Private Vulnerability Reporting:

- [Report a vulnerability](https://github.com/evanschleret/laravel-typebridge/security/advisories/new)

If private reporting is unavailable, contact the maintainer through GitHub and share details privately.

When reporting, include:

- Affected version
- Reproduction steps
- Impact assessment
- Proposed fix (if available)

You can expect an initial response as soon as possible.
